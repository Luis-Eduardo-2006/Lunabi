Carpeta de imágenes de productos.

Coloca aquí las fotos de cada producto. Formato recomendado: webp
cuadrado (800×800 px) con fondo blanco/beige, < 150 KB.

Nomenclatura sugerida por slug del producto (ver assets/js/data.js):
  glow-deep-serum-rice-arbutin-1.webp
  glow-deep-serum-rice-arbutin-2.webp
  advanced-snail-96-mucin-power-essence-1.webp

Actualmente data.js usa URLs de Unsplash como placeholder. Para usar
archivos locales, reemplaza los campos `imagenes[]` en cada producto
por rutas relativas, por ejemplo:
  imagenes: [
    "img/productos/glow-deep-serum-1.webp",
    "img/productos/glow-deep-serum-2.webp"
  ]
